# AI Toolbox

This repository is the foundation for a reusable AI-assisted development workflow.

## Purpose
The goal is to create a structured environment for terminal-based AI agents such as Claude Code, OpenCode, and Gemini CLI. The workflow reduces context bloat, preserves project memory, and forces disciplined execution through tasks, rules, hooks, and verification.

## Initial scope
This repository starts with the minimal core:
- `README.md` for the human-facing setup guide
- `AGENT.md` for the AI-facing workflow contract
- `.agent/memory/` for durable project context
- `.agent/rules/` for hard constraints
- `.agent/scripts/` for automation and hooks
