# 🏗️ Planning & Brainstorming

**When to use:**  
You are starting a completely new project, a large feature, or a complex system component. You need the AI to define architecture constraints *before* it starts writing a single line of code.

---

### English Prompt 🇺🇸
> "I want to build a [describe project/feature]. According to our AGENT.md rules, do not write code yet. First, analyze this request, identify constraints, and propose 2-3 architectural approaches for brainstorming. Wait for my decision before creating any tasks."

### German Prompt 🇩🇪
> "Ich möchte ein [Projekt/Feature] bauen. Gemäß unseren AGENT.md Regeln, schreibe bitte noch keinen Code. Analysiere diese Anfrage zuerst, identifiziere Einschränkungen und schlage 2-3 architektonische Ansätze für ein Brainstorming vor. Warte auf meine Entscheidung, bevor du Tasks erstellst."

### Russian Prompt 🇷🇺
> "Я хочу создать [проект/фичу]. Согласно нашим правилам в AGENT.md, пока не пиши код. Сначала проанализируй этот запрос, определи ограничения и предложи 2-3 архитектурных подхода для мозгового штурма. Подожди моего решения перед созданием задач."
