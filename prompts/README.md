# AI Toolbox Prompts

This directory contains battle-tested terminal prompts specifically designed to trigger the correct workflows within the AI Toolbox environment. 

Use these prompts instead of generic requests to significantly reduce hallucinations, enforce architecture decisions, and keep your AI Agent structured.

## Workflow Categories:
1. **[01-planning.md](01-planning.md)** - For new projects, major features, or brainstorming.
2. **[02-execution.md](02-execution.md)** - For picking up where you left off (the daily autopilot).
3. **[03-debugging.md](03-debugging.md)** - For structured bug hunting without breaking things.
4. **[04-refactoring.md](04-refactoring.md)** - For safe restructuring that respects your contracts.
