# Examples

This directory can hold sample workflows and execution traces for the AI Toolbox setup to demonstrate how an agent interacts with `rtk`, `Beads`, and the repository's hooks.

## Available Examples

1. **[ensemble-walkthrough.md](ensemble-walkthrough.md)** — End-to-end example: Building a REST API endpoint. Shows how all 5 tools (rtk, Beads, Superpowers, Templates, MCP) work together automatically.
2. **[parallel-review.md](parallel-review.md)** — Real example: 3-agent parallel code review of bootstrap scripts.
3. **[fresh-clone-setup.md](fresh-clone-setup.md)** — Step-by-step setup guide for initializing AI Toolbox in a new project.
